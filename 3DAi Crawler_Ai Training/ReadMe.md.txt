

# 3D AI Web Crawler and Model Training Pipeline

This project is a web crawler designed to scrape data from 3D model-related websites, process the crawled text using NLP techniques, and simulate the training of an AI model. It uses Docker to deploy a Flask app, Celery for background tasks, and Redis for message brokering.

## Prerequisites

- Docker and Docker Compose installed on your machine.
